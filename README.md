# Транскрибация звонков и контроль их качества.

## Описание проекта
 Cоздать и настроить систему для транскрибации звонков в отделе продаж и контроля их качества. 
Интересная и полезная для бизнеса тема, с его помощью легко анализировать проведённые разговоры с клиентами. Это сервис, который будет слушать звонки менеджеров и с помощью искусственного интеллекта на основании этих звонков сделает организационные выводы. 
Это очень востребованная функция для бизнеса с его помощью легко анализировать проведённые разговоры с клиентами.
Эти сценарии можно интегрировать в CRM-систему, создавая оповещения, алармы и другие уведомления.
Это позволяет повысить конверсию за счёт определения потребностей клиента. Например, вставив в скрипт менеджера открытый вопрос: "Что вам мешает совершить покупку прямо сейчас?" и получив 20 ответов клиентов, пять из которых созвучны, можно понять проблему и решить её.

Теперь каждый обработанный разговор будет отправляться в Google-таблицу в удобном для чтения формате.

###  Создан в Make через Console Gmail API.

 Make — визуальная платформа для автоматизации рабочих процессов и интеграции онлайн-сервисов и приложений без написания кода (no-code/low-code). Позволяет создавать сложные сценарии взаимодействия между системами.
